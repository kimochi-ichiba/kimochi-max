## ⚡ 4つの新機能

### 1. 段階的レバレッジ (dynamic lev multi-step)

| 条件 | レバレッジ |
|---|---|
| 資産が初期の 2倍未満 | 1.5x |
| 2倍 〜 5倍 | 2.0x |
| 5倍 〜 10倍 | 2.5x |
| 10倍以上 | 3.0x |

→ 資産が増えるほど攻める、少ない時は守る設計

### 2. 積立自動投入

- 毎月1日に指定額をboard内に追加
- `state.json` の `added_capital_history` に記録
- `total_equity` = `virtual_market_value` + `locked_bank` + `added_capital`
- リバランス時に積立額も含めて比率計算

### 3. 目標到達時の段階利確

| マイルストーン | 発動行動 |
|---|---|
| 資産 5,000万円到達 | **25%を利確** → USDT枠へ移動 |
| 資産 1億円到達 | **35%を利確** → USDT枠へ移動 |
| 資産 1.5億円到達 | **50%を利確** → 「日本円逃避予定」にマーク |
| 35歳到達 | **残り全額を利確**、VYM移行モードへ |

### 4. 年度最大損失ストップ

- 年初資産を記録、年度で -30% を超えたらボット停止
- ユーザーに Discord / Slack 通知
- 手動復帰操作必要

## ⚠️ リスク管理

### 想定リスクと対策

| リスク | 確率 | 対策 |
|---|---|---|
| BTC暴落 (-60%以上) | 中 | 年度最大損失ストップ |
| 取引所倒産 | 低 | 複数取引所に分散、大部分はUSDT |
| 規制変更 | 中 | 2027年までに税制確認、移行計画修正 |
| ボット不具合 | 低 | 日次バックアップ、人間監視 |
| 積立挫折 | 高 | 自動化で感情を排除 |

### 不安要素の解消

1. **「バックテストと実運用のギャップ」**
   → 小額で1-3ヶ月SIMテスト後、段階的に本運用

2. **「2028年の分離課税20%が本当に通るか」**
   → 通らなくても総合課税での試算を並行表示

3. **「4年サイクルでの大暴落」**
   → 32-33歳のバブル時に自動で部分利確
